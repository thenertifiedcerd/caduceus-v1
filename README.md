# Caduceus V1

Caduceus is a health and workout tracking app built with React, Vite, and Firebase.

## Stack

- React 19
- Vite
- React Router
- React Bootstrap
- Firebase Auth and Firestore

## Development

```bash
npm install
npm run dev
```

## Environment

Copy `.env.example` to `.env` and add your Firebase configuration values before running the app.
